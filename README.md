# file-copy-system-calls
Утилита для копирования файлов на языке C с использованием системных вызовов. Реализует три режима работы: копирование с большим буфером, с буфером 32 байта и с сохранением прав доступа для исполняемых файлов.


**Функциональность:**
1. Базовый функционал (6 баллов): Программа читает и записывает
файлы используя системные вызовы ( open , read , write , close , stat ).
Используется буфер, размер которого превышает размер читаемого
файла.
2. Опционально +1 балл: Реализовано копирование с буфером
ограниченного размера (32 байта) с циклическим чтением и записью.
3. Опционально +1 балл: Реализовано сохранение прав доступа для
исполняемых файлов. Программа определяет, является ли файл
исполняемым и сохраняет соответствующие права доступа. Для
обычных текстовых файлов устанавливаются стандартные права 0644.


**Инструкция по запуску:**


gcc -o copy_file copy_file.c


echo "Это простой текстовый файл для тестирования" > test.txt

--------


echo '#!/bin/bash


echo "Это исполняемый скрипт"


ls -la' > script.sh

---------


chmod +x script.sh


**Запуск тестов:**


./copy_file test.txt copy1.txt


./copy_file test.txt copy2.txt -small


./copy_file script.sh script_copy.sh -preserve
